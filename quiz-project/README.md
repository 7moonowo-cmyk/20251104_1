# 測驗系統

這是一個簡單的測驗系統，使用 JavaScript 和 HTML 建立，旨在提供用戶一個互動式的測驗體驗。

## 文件結構

- `src/quiz.js`: 包含測驗系統的主要邏輯，負責載入問題、顯示問題、管理用戶答案以及顯示結果。還包括在問題之間導航和更新 UI 的功能。
- `src/common.js`: 包含項目中使用的共享函數和工具，例如處理選單顯示和其他常見功能。
- `src/questions.csv`: 包含測驗問題及其對應選項和答案的 CSV 格式文件。
- `index.html`: 主 HTML 文件，設置測驗應用程序的結構。包括對 JavaScript 文件的引用並提供測驗界面的初始佈局。
- `package.json`: npm 的配置文件，列出項目的依賴和腳本。

## 使用說明

1. 確保您已安裝 Node.js 和 npm。
2. 克隆此項目到本地計算機。
3. 在項目根目錄中運行 `npm install` 以安裝所需的依賴。
4. 使用本地 HTTP 伺服器（例如：在項目資料夾執行 `python -m http.server 8000`）來啟動應用程序。
5. 打開瀏覽器並訪問 `http://localhost:8000` 開始測驗。

## 功能

- 隨機選擇問題進行測驗。
- 顯示用戶的得分和正確答案。
- 提供問題的詳細解析。
